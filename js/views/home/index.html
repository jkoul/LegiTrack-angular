Welcome to LegiTrack DC!!!

About

LegiTrack was inspired by the OpenLIMS project created by V. David Zvenyach. It provides a responsive and flexible interface to DC's legislative information.

How to Use

  1. View All Bills: click on the "All Bills" option under the "Browse Legislation" tab.
  2. Search Bills: enter any word in the Search bar in the header. The list of bills will update automatically.
  3. View a Bill: Click on the title of any bill to view more details
